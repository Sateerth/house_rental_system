Generated at 2025-11-09T18:10:50.560230Z

## Setup
1. Create a virtualenv: `python -m venv venv`
2. Activate it and install requirements: `pip install -r requirements.txt`
3. Run: `python app.py`
4. App will be available at http://127.0.0.1:5000/

## Features
- Add houses, tenants, bills, and agreements.
- Owner dashboard aggregates bills per house.
